    2. JavaScript został stworzony w 1995 roku.

  2. 2)dlaczego jest to język skryptowy

    3. JavaScript jest językiem skryptowym, ponieważ jest interpretowany podczas działania, a nie kompilowany przed uruchomieniem programu.

  3. 3)gdzie jest wykonywany JS (klient lub serwer)?

    4. Po stronie klienta, JavaScript jest używany do tworzenia interaktywnych stron internetowych i obsługi zdarzeń w przeglądarkach. Po stronie serwera, może być używany do tworzenia aplikacji serwerowych i interakcji z bazą danych..

  4. 4)dlaczego jest JS to bezpieczne narzędzie

    5. JavaScript nie jest uważany za narzędzie o wysokim poziomie bezpieczeństwa

  5. 5)czy jest to język obiektowy , jeśli tak to dlaczego.

    6. Tak, JavaScript jest językiem obiektowym. Oznacza to, że w JavaScript obiekty odgrywają kluczową rolę, a większość wartości jest obiektami lub może być traktowana jako obiekty

  6. 6)wymień zalety JS:

  1. Zaleta 1: Wysoka dostępność - JavaScript jest wbudowany w większość przeglądarek internetowych, co oznacza, że jest szeroko dostępny na różnych platformach.
  2. Zaleta 2: Interaktywność - JavaScript umożliwia tworzenie interaktywnych elementów na stronach internetowych, co zwiększa zaangażowanie użytkowników.
  3. Zaleta 3: Wieloplatformowość - JavaScript można używać zarówno po stronie klienta, jak i po stronie serwera, co sprawia, że jest uniwersalnym językiem programowania.

    4. Aplikacje responsywne to aplikacje lub strony internetowe, które dostosowują się do różnych rozmiarów ekranów i urządzeń.

  4. 10)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu

    • type=button: Określa, że jest to przycisk formularza.
    • name="przycisk": Nadaje przyciskowi nazwę "przycisk". Ta nazwa może być używana do identyfikowania przycisku w skrypcie JavaScript lub w formularzu, jeśli istnieje.
    • value="Nowa Strona": Określa tekst, który będzie widoczny na przycisku. Tekst ten będzie wyświetlany na przycisku jako "Nowa Strona".
    • onclick="WinOpen(' '): Jest to atrybut onclick, który określa kod JavaScript do wykonania, gdy użytkownik kliknie przycisk. W tym przypadku, po kliknięciu przycisku zostanie wywołana funkcja WinOpen(' ').
    • WinOpen jest potencjalnie funkcją JavaScript, która może zostać wywołana po kliknięciu przycisku.

    • 11)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

      • window.open(): Jest to wbudowana funkcja przeglądarki, która służy do otwierania nowego okna przeglądarki lub nowej karty.
      • "obraz.html": To pierwszy argument funkcji i określa adres URL, który zostanie załadowany w nowym oknie lub karcie. W tym przypadku, otwierane jest okno z adresem "obraz.html".
      • "okienko": To drugi argument i określa nazwę nowego okna lub karty. Jeśli okno o tej nazwie nie istnieje, zostanie utworzone nowe. Ta nazwa może być używana do manipulacji tym oknem w przyszłości.
      • toolbar=no: Wyłącza pasek narzędzi w nowym oknie.
      • directories=no: Wyłącza pasek z linkami do katalogów.
      • menubar=no: Wyłącza pasek menu.
      • height=280: Określa wysokość nowego okna (280 pikseli).
      • width=160: Określa szerokość nowego okna (160 pikseli).
      • top=200: Określa pozycję okna od góry ekranu (200 pikseli od góry).
      • left=200: Określa pozycję okna od lewej strony ekranu (200 pikseli od lewej).
